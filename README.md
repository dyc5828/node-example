## Get Started

1. [Download](https://nodejs.org/en/download/) and install Node.js 

2. Check `node` and `npm` version

```shell
$ node -v // 3.x.x
$ npm -v  // v6.x.x
```

2. Install `nodemon` globally

```shell
$ [sudo] npm install --global nodemon
```

3. Install local node modules

```shell
$ [sudo] npm install
```

4. Run with app restarts

```shell
$ nodemon [index.js]
```
